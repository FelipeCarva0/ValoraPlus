# 📌 Opções de Investimento

Sistema web desenvolvido com o objetivo de auxiliar usuários na escolha de investimentos adequados ao seu perfil financeiro, oferecendo recomendações personalizadas e informações sobre diferentes tipos de aplicações.

---

## 🎯 Objetivo

O projeto visa apoiar usuários na tomada de decisão financeira, permitindo:

- Identificação do perfil de investidor (conservador, moderado ou agressivo)
- Sugestão de opções de investimento
- Promoção de educação financeira básica

---

## ⚙️ Funcionalidades

- Questionário para definição do perfil do investidor  
- Exibição de opções de investimento  
- Interface interativa para navegação das informações  

---

## 🛠️ Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript  
- Node.js
